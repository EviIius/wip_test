Developed a system for chunking data from multiple documents to enhance the reference materials for a Large Language Model (LLM).
The project was motivated by challenges related to environmental impact in online compute platforms.
Enables the model to search a document database to retrieve relevant information for generating responses.
Designed to integrate with Hugging Face's chat template, utilizing relevance scoring to determine the most accurate text source for responses.
The next phase of this project would involve implementing ReAct (Reason + Action) prompting, if time permitted.
